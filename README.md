## 📚 Sobre o Curso

Curso de capacitação tecnológica para formação de pessoas aptas a atuar em empresas do Estado de Santa Catarina, independentemente do local onde estejam instaladas.

O curso é dividido em **três módulos**:

### Módulo I
⏱️ **04 horas**  
Conceitos de Lógica da Programação

### Módulo II
⏱️ **200 horas**  
Programador de Sistemas e Desenvolvimento Web com IA

### Módulo III
⏱️ **240 horas**  
Desenvolvimento Web com IA
